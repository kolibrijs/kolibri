# create-kolibri

[![NPM version](https://img.shields.io/npm/v/kolibrijs?color=9945FF&label=)](https://www.npmjs.com/package/create-kolibri)

Starter template generater for [Kolibri](https://kolibri.show).

## Usage

```bash
npm init kolibri
```

or

```bash
yarn create kolibri
```
