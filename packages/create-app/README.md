# create-kolibry

[![NPM version](https://img.shields.io/npm/v/create-kolibry?color=9945FF&label=)](https://www.npmjs.com/package/create-kolibry)

Starter template generater for [Kolibry](https://kolibry.dev).

## Usage

```bash
npm init kolibry
```

or

```bash
yarn create kolibry
```
